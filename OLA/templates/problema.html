Problemas Aleatórios do Autor

Você deve criar uma estrutura de sistema Web (em Django) contendo os elementos abaixo (além dos elementos existentes e criados automaticamente pelo framework Django):

  • <nome do modulo>/
  • models.py (o(s) seu(s) modelo(s))
  • views.py (suas visões)
  • templates/
    • index.html (apresenta o seu projeto)
    • problema.html (descreve detalhadamente seu problema)
    • solucao.html (descreve como a solução deverá resolver o problema)
    • autor.html (mostra seu nome com um minicurrículo)

Crie um repositório no GitHub e faça:

  1. Crie uma nova branch chamada “organizacao”
  2. Crie, programe e envie seus códigos (commit and push)
  3. Teste todo o funcionamento e certifique-se de que está tudo funcionando
  4. Mescle a branch “organizacao” com a ramificação principal
